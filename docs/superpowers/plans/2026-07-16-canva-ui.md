# Canva UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce an editable Canva design with four consistent 360 x 780 HarmonyOS app screens for 下一件事.

**Architecture:** Use a single 1440 x 780 Canva canvas containing four adjacent 360 x 780 artboards. Establish reusable visual tokens and shared navigation first, then compose each screen with editable Canva text, shapes, and line icons.

**Tech Stack:** Canva editor; editable text, rectangles, circles, lines, and built-in line icons.

---

### Task 1: Set Up The Design Canvas And Shared Tokens

**Files:**
- Reference: `docs/superpowers/specs/2026-07-16-canva-ui-design.md`

- [ ] **Step 1: Create the single Canva design canvas**

Create a custom-size Canva design at `1440 x 780 px`. Add four 360 x 780 page regions in this order: Now, Focus Timer, Clear, Today Review.

- [ ] **Step 2: Apply the shared visual tokens**

Use `#FAF8F2` for ordinary page backgrounds; `#FFFFFF` for cards; `#252525` and `#777777` for text; `#E8E6E0` for borders; `#4FA36C`, `#7A9CC6`, and `#C9855B` only as muted accents. Use Chinese sans-serif typography, 12-16 px card radii, and minimal shadows.

- [ ] **Step 3: Create the reusable bottom navigation**

Add a bottom navigation with the three labels “现在”, “清空”, and “回顾” and simple line icons. Duplicate it for the three light screens and activate exactly one item on each screen; omit it from Focus Timer.

### Task 2: Compose The Now And Focus Timer Screens

**Files:**
- Reference: `docs/superpowers/specs/2026-07-16-canva-ui-design.md`

- [ ] **Step 1: Build the Now screen**

Add “今天已专注 20 分钟，完成 2 次”, “现在只做这一件事”, a card titled “写数学作业的第一题”, a 5 / 10 / 25 minute segmented control with 25 minutes selected, and a full-width green “开始专注” button. Activate “现在” in navigation.

- [ ] **Step 2: Build the Focus Timer screen**

Use full-screen `#18201C`. Add the task name, a thin circular green progress ring around `24:58`, the copy “先做一小步就好”, a primary pause/resume icon button, and an outlined “提前结束” button. Do not add navigation, task lists, or statistics.

### Task 3: Compose The Clear And Today Review Screens

**Files:**
- Reference: `docs/superpowers/specs/2026-07-16-canva-ui-design.md`

- [ ] **Step 1: Build the Clear screen**

Add “脑子里有什么，先写下来”, “不用整理，先放在这里”, an input with “想到什么就写什么” and a send/add icon, then rows for “作业还没有写”, “晚上要洗衣服”, “回复朋友消息”, and “想学鸿蒙开发”. Each row includes “设为下一件事” and a delete icon. Activate “清空” in navigation.

- [ ] **Step 2: Build the Today Review screen**

Add “今日回顾”, side-by-side summaries “总专注 35 分钟” and “完成 3 次”, and the rows: “20:30 写数学作业的第一题 10 分钟 已完成”, “18:10 收拾书桌 5 分钟 已完成”, and “16:00 背英语单词 20 分钟 未完成”. Use muted green for completed and muted orange for incomplete. Activate “回顾” in navigation.

### Task 4: Perform Visual Quality Review

**Files:**
- Reference: `docs/superpowers/specs/2026-07-16-canva-ui-design.md`

- [ ] **Step 1: Check Chinese text and touch targets**

At 100% zoom, inspect every Chinese label. Correct truncation, overlap, or cramped spacing; ensure buttons, segmented controls, navigation items, and per-row actions have phone-appropriate touch areas.

- [ ] **Step 2: Check the shared design language**

Verify the three light screens have identical navigation placement, typography hierarchy, border color, card radii, and icon style. Verify the dark focus screen retains the same restrained typography and muted accent treatment.

- [ ] **Step 3: Check scope and editability**

Confirm all visible words are Canva text layers and all UI is built from editable native shapes or built-in icons. Remove phone frames, gradients, marketing elements, achievement systems, complex to-do controls, and decorative illustrations.
