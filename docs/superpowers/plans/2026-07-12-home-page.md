# Home Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the template welcome screen with a task entry home page that selects 5, 10, or 25 minutes and rejects an empty task.

**Architecture:** Keep page-local UI state in `Index.ets`; move the trim-and-validate rule into `home/HomePageValidation.ets` so it remains independent of ArkUI widgets. Render validation and confirmation feedback in the same page because this feature intentionally has no timer route.

**Tech Stack:** HarmonyOS API 24, ArkTS, ArkUI, Hypium test source, Hvigor.

- [ ] **Step 3: Implement the minimal validation module**

```ts
export interface TaskValidationResult {
  isValid: boolean;
  taskTitle: string;
}

export function validateTaskTitle(taskTitle: string): TaskValidationResult {
  const normalizedTitle = taskTitle.trim();
  return { isValid: normalizedTitle.length > 0, taskTitle: normalizedTitle };
}
```

### Task 2: Implement the Home Page

**Files:**
- Modify: `NextThing/entry/src/main/ets/pages/Index.ets`

- [ ] **Step 1: Add page state**

```ts
@State taskTitle: string = '';
@State selectedMinutes: number = 25;
@State feedback: string = '';
```

- [ ] **Step 2: Replace the template text with the input, duration controls, start button, and feedback**

```ts
TextInput({ placeholder: '请输入下一件事', text: this.taskTitle })
  .onChange((value: string) => { this.taskTitle = value; })

Button('25 分钟').onClick(() => { this.selectedMinutes = 25; })

Button('开始').onClick(() => {
  const result = validateTaskTitle(this.taskTitle);
  this.feedback = result.isValid
    ? `已准备开始：${result.taskTitle}（${this.selectedMinutes} 分钟）`
    : '请输入任务内容';
})
```
